天气预报数据处理#

备注

标题中所谓的"处理", 指的是对天气预报数据的获取, 解析与存储操作.

本文档对售电交易平台中所用到的天气预报数据处理模块进行了说明. 数据来源及获取方式分别为:

  • 和风气象: 通过和风天气API定时获取获取

  • 天机数据: 签订数据服务协议, 对方定时推送数据至目标服务器.
