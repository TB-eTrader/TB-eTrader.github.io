2、中长期交易量价费汇总表解析
----------------------------------------

在 **项目根目录** 下执行以下命令以解析目标路径下的所有PDF文件：

.. code-block:: bash

    python -m summary_sheet_parse.schedule_run --pdf_dir <pdf_dir>

其中 ``<pdf_dir>`` 是存放pdf文件的目录，可以是绝对路径，也可以是基于项目根目录的相对路径。

更多可选的参数及其用法可以通过 ``python -m summary_sheet_parse.schedule_run -h`` 或 :func:`~.schedule_run.parse_args` 查看。
